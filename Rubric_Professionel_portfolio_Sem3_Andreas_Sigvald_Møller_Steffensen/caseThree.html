codeacademy Challenge: Prime Number Finder

Challenge

The challenge given by codeacademy was to make a function that would take in a number and find all the prime numbers between 1 and the given number. The return from the function should be an array with all prime numbers listed.

Solution

A piece of JavaScript that returns all the prime numbers between 1 and the given number in an array, that will be displayed in the console.

Roles

Frontend developer

Solution

A piece of JavaScript code that takes a given number and displays all prime numbers in between 1 and the given number.

Code

For my solution, i use 2 for-loops and 1 if statement. The first for-loop is used to loop through 1 and the given number. Then the numbers goes through the next for-loop, which checks if the number is divisible and if they aren't, that means it is a prime number.

Then the if statement prints the numbers if they were found to be prime numbers
